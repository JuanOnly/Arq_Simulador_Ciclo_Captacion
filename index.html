CPU Fetch Cycle Simulator

What is this? / ¿Qué es esto?

This is an interactive simulator that teaches how a CPU fetches and executes instructions. It demonstrates the Fetch Cycle, the fundamental process by which computers retrieve instructions from memory.

Este es un simulador interactivo que enseña cómo una CPU captura y ejecuta instrucciones. Demuestra el Ciclo de Captación, el proceso fundamental por el cual las computadoras recuperan instrucciones de la memoria.

How it works / Cómo funciona

Every instruction your computer runs goes through two main phases:

1. FETCH → 2. EXECUTE

Cada instrucción que tu computadora ejecuta pasa por dos fases principales:

1. CAPTACIÓN → 2. EJECUCIÓN

Operation Code (4 bits): Tells the CPU what to do (add, subtract, move data, etc.)
Address 1 (6 bits): Location of first data (0-63)
Address 2 (6 bits): Location of second data (0-63)
